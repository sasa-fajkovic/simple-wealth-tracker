> All decisions sourced from ROADMAP.md, REQUIREMENTS.md, and codebase scan ‚Äî no user input required.

| Property | Value |
|----------|-------|
| Tool | none |

## Spacing Scale

Declared values (all multiples of 4, expressed as Tailwind tokens):

| Token | Tailwind Class | px Value | Usage |
|-------|---------------|----------|-------|
| xs | `gap-1` / `p-1` | 4px | Icon gaps, color-dot margin, pill inner spacing supplement |
| sm | `gap-2` / `p-2` | 8px | Pill horizontal padding, icon button padding, inline element gaps |
| md | `gap-4` / `p-4` | 16px | Card internal padding horizontal, chart container padding |
| lg | `gap-6` / `p-6` | 24px | Card internal padding vertical, section spacing between cards and chart |
| xl | `gap-8` / `p-8` | 32px | Page horizontal padding on sm+ screens |
| 2xl | `gap-12` | 48px | Major section breaks (cards-to-chart gap on desktop) |
| 3xl | `gap-16` | 64px | Page-level top padding |

**Exceptions:**
- `WealthChart` parent div: `h-[420px]` ‚Äî explicit fixed height, required by Recharts `ResponsiveContainer` to prevent zero-height collapse. This is a Recharts constraint, not a spacing token.
- Nav bar height: `h-14` (56px) ‚Äî standard top nav target height.
- Touch targets (hamburger, chart-type buttons): minimum `44px` tap area via `p-2` on `h-10 w-10` containers.

## Typography

All sizes from Tailwind's default type scale. **Two weights maximum.**

| Role | Tailwind Class | px | Weight | Line Height | Usage |
|------|---------------|-----|--------|-------------|-------|
| Display | `text-2xl font-bold` | 28px | 700 | 1.1 (`leading-tight`) | Total Net Worth value |
| Heading | `text-lg font-medium` | 18px | 500 | 1.2 (`leading-snug`) | Card titles ("Total Net Worth", "Period Change", "Category Breakdown"), page section labels |
| Body | `text-sm font-medium` | 14px | 500 | 1.5 (`leading-normal`) | Category breakdown rows, tooltip content, nav links, period change amount |
| Label | `text-xs font-medium` | 12px | 500 | 1.4 (`leading-normal`) | Card eyebrow labels ("TOTAL NET WORTH"), range pill text, chart-type button aria-labels, percentage badges |

> **Weight contract:** Exactly 2 weights ‚Äî `font-bold` (700) for Display values only; `font-medium` (500) for all other text roles. Do not introduce additional weights.

## Interaction Contracts

### 1. Range Pill Selection

- **Trigger:** User clicks a pill label (`YTD`, `6M`, `1Y`, etc.)
- **State change:** Previously active pill ‚Üí inactive style; clicked pill ‚Üí active style
- **Side effect:** Calls `getSummary(range)` and re-renders `WealthChart` + `SummaryCards` with new data
- **Loading:** During fetch, chart area shows a loading overlay (opacity-50 + spinner centered); pills remain clickable but new clicks are debounced until response resolves
- **Inactive pill style:** `bg-white border border-gray-200 text-gray-600 hover:bg-gray-50 rounded-full px-3 py-1 text-xs font-medium transition-colors`
- **Active pill style:** `bg-blue-600 border border-blue-600 text-white rounded-full px-3 py-1 text-xs font-medium`
- **Default on mount:** `1Y` is pre-selected

### 2. Chart Type Toggle (ChartTypeSelector)

- **Trigger:** User clicks one of three icon buttons (Area / Line / Bar)
- **Persistence:** Reads initial value from `localStorage` key `dashboard-chart-type`; writes on every change. Default when key absent: `"area"`
- **Storage values:** `"area"` | `"line"` | `"bar"`
- **Inactive button style:** `p-2 rounded border border-gray-200 bg-white text-gray-500 hover:bg-gray-50 transition-colors`
- **Active button style:** `p-2 rounded border border-blue-600 bg-blue-600 text-white`
- **Icons:** `AreaChart` / `LineChart` / `BarChart2` from `lucide-react`, size `16px`
- **Container:** `flex gap-1` aligned to the top-right of the chart card header

### 3. Recharts Chart (`WealthChart`)

- **Parent container:** `<div className="h-[420px]">` ‚Äî mandatory, prevents zero-height collapse
- **Chart component:** `<ComposedChart>` inside `<ResponsiveContainer width="100%" height="100%">`
- **Stacked Area mode:** One `<Area>` per category, all with `stackId="wealth"`, `dataKey={category.category_id}`, `fill={category.color}`, `stroke={category.color}`, `fillOpacity={0.6}`
- **Line mode:** One `<Line>` per category, `dataKey={category.category_id}`, `stroke={category.color}`, `dot={false}`, `strokeWidth={2}`
- **Stacked Bar mode:** One `<Bar>` per category, all with `stackId="wealth"`, `dataKey={category.category_id}`, `fill={category.color}`
- **Total wealth line (all modes):** One `<Line>` with `dataKey="total"` (sourced from `totals[]` mapped to each data point as `{ month, total, [category_id]: value, ... }`), `stroke="#111827"`, `strokeWidth={2}`, `dot={false}`, no `stackId` (not stacked ‚Äî absolute value overlay). Rendered after all per-category series so it paints on top. Legend label: `Total`.
- **X-axis:** `<XAxis dataKey="month" tickFormatter={(m) => m.slice(0, 7)} tick={{ fontSize: 12, fill: '#6B7280' }} />`
- **Y-axis:** `<YAxis tickFormatter={(v) => new Intl.NumberFormat('de-DE', { notation: 'compact', maximumFractionDigits: 0 }).format(v)} tick={{ fontSize: 12, fill: '#6B7280' }} />`
- **Tooltip:** Custom `<Tooltip content={<CustomTooltip />}>` ‚Äî white bg, `border border-gray-200`, `rounded-lg shadow-lg p-3 text-sm`; lists each category name + formatted EUR value; final row shows bold "Total" + formatted EUR value
- **Grid:** `<CartesianGrid strokeDasharray="3 3" stroke="#E5E7EB" />`
- **Legend:** `<Legend wrapperStyle={{ fontSize: '12px', color: '#374151' }}>`

### 4. Mobile Navigation (<640px)

- **Breakpoint:** Tailwind `sm` breakpoint (640px)
- **Desktop (‚â•640px):** Nav links displayed inline `flex gap-6`; hamburger hidden (`hidden sm:flex`)
- **Mobile (<640px):** Nav links hidden (`hidden`); hamburger icon button visible
- **Hamburger icon:** `Menu` (closed) / `X` (open) from `lucide-react`, `size={20}`
- **Dropdown panel:** `absolute top-14 left-0 right-0 bg-white border-b border-gray-200 z-50`; links stacked `flex flex-col py-2`; each link `px-4 py-3 text-sm font-medium`
- **Dismiss:** Clicking a nav link closes the dropdown; clicking outside closes the dropdown (click-outside handler)
- **Active state in dropdown:** Same as desktop ‚Äî `text-blue-600 font-medium`

### 5. Loading State

- **Initial load:** Dashboard renders two states in sequence: (a) skeleton placeholders for SummaryCards (3 gray animated `animate-pulse` boxes `h-24 bg-gray-100 rounded-lg`), (b) skeleton for chart area (`h-[420px] bg-gray-100 rounded-lg animate-pulse`)
- **Range change re-fetch:** Chart area overlays `opacity-60` with centered spinner `animate-spin`; cards update after data arrives

### 6. Error State

- **Trigger:** `getSummary()` throws `ApiError` (non-2xx)
- **Presentation:** Full-width `bg-red-50 border border-red-200 rounded-lg p-4` alert above the chart, containing heading `"Could not load data"` in `text-red-700 font-medium` and body copy in `text-red-600 text-sm`
- **Recovery:** `"Retry loading data"` button within the error banner, re-triggers the fetch
